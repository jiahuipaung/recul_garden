在 Go 语言中解决 LeetCode 问题时，熟悉以下常用库函数和操作可以显著提升编码效率。以下是分类整理的常用工具和示例：

### 一、**字符串处理（`strings` 包）**
1. **分割与连接**：
   - `strings.Split(s, sep string) []string`：按分隔符分割字符串。
   - `strings.Join(slice []string, sep string) string`：将字符串切片连接成一个字符串。
2. **子串判断**：
   - `strings.Contains(s, substr string) bool`：判断是否包含子串。
   - `strings.HasPrefix(s, prefix string) bool` / `strings.HasSuffix(s, suffix string) bool`：判断前缀或后缀。
3. **索引与统计**：
   - `strings.Index(s, substr string) int`：返回子串首次出现的索引。
   - `strings.Count(s, substr string) int`：统计子串出现次数。

**示例**：
```go
s := "hello-world"
parts := strings.Split(s, "-")      // ["hello", "world"]
joined := strings.Join(parts, "_")  // "hello_world"
contains := strings.Contains(s, "lo") // true
```

---

### 二、**类型转换（`strconv` 包）**
4. **字符串与数字**：
   - `strconv.Atoi(s string) (int, error)`：字符串转整型。
   - `strconv.Itoa(i int) string`：整型转字符串。
5. **进制转换**：
   - `strconv.FormatInt(n int64, base int) string`：将整数转换为指定进制的字符串。

**示例**：
```go
numStr := "123"
num, _ := strconv.Atoi(numStr)      // 123
str := strconv.Itoa(456)            // "456"
binary := strconv.FormatInt(10, 2)  // "1010"
```

---

### 三、**排序（`sort` 包）**
6. **基本排序**：
   - `sort.Ints(slice []int)`：对整数切片排序。
   - `sort.Strings(slice []string)`：对字符串切片排序。
7. **自定义排序**：
   - 使用 `sort.Slice(slice, func(i, j int) bool)`，通过闭包定义比较逻辑。

**示例**：
```go
nums := []int{3, 1, 4}
sort.Ints(nums) // [1, 3, 4]

people := []struct{ Name string; Age int }{{"Bob", 25}, {"Alice", 30}}
sort.Slice(people, func(i, j int) bool { return people[i].Age < people[j].Age })
```

### 四、**数学运算（`math` 包）**
8. **常用函数**：
   - `math.Max(x, y float64) float64` / `math.Min(x, y float64) float64`（注意：需转为 `float64`）。
   - `math.Abs(x float64) float64`：绝对值。
   - `math.Pow(x, y float64) float64`：计算 `x^y`。
9. **取整与舍入**：
   - `math.Floor(x float64) float64`（向下取整） / `math.Ceil(x float64) float64`（向上取整）。

**示例**：
```go
maxVal := math.Max(3.0, 5.0) // 5.0
abs := math.Abs(-10)         // 10.0
power := math.Pow(2, 3)      // 8.0
```

### 五、**切片操作**
10. **动态操作**：
   - `append(slice []T, elems ...T) []T`：添加元素。
   - `copy(dst, src []T) int`：复制切片，避免共享底层数组。
11. **初始化与容量**：
   - `make([]T, len, cap int) []T`：创建指定长度和容量的切片。

**示例**：
```go
slice := []int{1, 2}
slice = append(slice, 3) // [1, 2, 3]
newSlice := make([]int, len(slice))
copy(newSlice, slice)    // 深拷贝
```

### 六、**堆与优先队列（`container/heap` 包）**
需实现 `heap.Interface`（定义 `Len()`, `Less()`, `Swap()`, `Push()`, `Pop()` 方法）。常用操作：
- `heap.Init(h heap.Interface)`
- `heap.Push(h, x interface{})`
- `heap.Pop(h interface{}) interface{}`

**示例**：
```go
type IntHeap []int
func (h IntHeap) Len() int { return len(h) }
func (h IntHeap) Less(i, j int) bool { return h[i] < h[j] }
func (h IntHeap) Swap(i, j int) { h[i], h[j] = h[j], h[i] }
func (h *IntHeap) Push(x interface{}) { *h = append(*h, x.(int)) }
func (h *IntHeap) Pop() interface{} {
    old := *h
    n := len(old)
    x := old[n-1]
    *h = old[:n-1]
    return x
}

h := &IntHeap{3, 1, 4}
heap.Init(h)
heap.Push(h, 2)
fmt.Println(heap.Pop(h)) // 1
```

## **什么时候需要用 `rune`？**

1. **处理 Unicode 字符（非 ASCII）**
    
    - 如中文、日文、韩文、特殊符号、Emoji 等。
    - `string[i]` 可能截取到 UTF-8 **编码字节** 而不是完整字符，导致错误。
2. **使用 `unicode` 标准库函数**
    
    - 例如：
        
        go
        
        复制编辑
        
        `unicode.IsLetter(rune(s[i]))  unicode.ToLower(rune(s[i]))`
        
    - 这些函数接受 `rune` 作为输入，而不是 `byte`。
3. **遍历字符串时避免 UTF-8 切割问题**
    
    - 推荐用 `for _, ch := range s {}`，其中 `ch` 是 `rune`，不会截断多字节字符。
